# Electronic_Projects

Repositorio de practica y consulta para proyectos relacionados con
programacion en C, C++ y Arduino.

## Estructura

| Carpeta | Tema | Enfoque |
| --- | --- | --- |
| `C/` | Lenguaje C | Fundamentos de programacion de bajo nivel, memoria y compilacion. |
| `C++/` | Lenguaje C++ | Programacion multiparadigma, clases, objetos, plantillas y STL. |
| `Arduino/` | Plataforma Arduino | Programacion de microcontroladores, sensores, actuadores y prototipos electronicos. |

## Objetivo

El objetivo del repositorio es organizar ejemplos, apuntes y ejercicios por
tecnologia. Cada carpeta debe contener su propio README con el contexto del
tema que le corresponde, de modo que cualquier persona pueda entender el
proposito de la carpeta sin depender de informacion externa o de conversaciones
previas.

## Recomendaciones de uso

- Mantener cada ejemplo dentro de la carpeta que corresponde a su tecnologia.
- Documentar dependencias, placas, compiladores o herramientas cuando sean
  necesarias.
- Evitar mezclar codigo de diferentes temas si no existe una razon clara.
- Actualizar el README de la carpeta cuando se agreguen conceptos, ejemplos o
  flujos nuevos.
